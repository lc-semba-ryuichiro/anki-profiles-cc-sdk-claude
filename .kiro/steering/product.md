# Product Overview

プロフィール帳暗記ツールは、人物情報(名前、顔写真、所属、役職)を効率的に記憶するためのWeb学習支援アプリケーションです。間隔反復学習アルゴリズム(Ankiスタイル)を用いて長期記憶の定着を支援します。

## Core Capabilities

- **プロフィール管理**: 名前、顔写真、所属、役職、メモを含む人物情報の登録・編集・削除
- **間隔反復学習**: SM-2アルゴリズムに基づくフラッシュカード学習機能
- **進捗可視化**: 学習統計、習得レベル、学習実績グラフによるモチベーション維持
- **データ永続化**: ブラウザローカルストレージ/IndexedDBによるオフライン対応とバックアップ機能

## Target Use Cases

- ビジネスシーンでの人物関係構築(顔と名前の記憶)
- 学習者向けの効率的な記憶定着支援
- プライバシー重視のローカルファースト学習環境

## Value Proposition

- **完全オフライン動作**: サーバー不要、ブラウザのみで完結するプライバシー保護
- **科学的学習手法**: 記憶の定着効率を最大化する間隔反復アルゴリズム採用
- **直感的UI**: レスポンシブデザイン、キーボードショートカット、アクセシビリティ対応
